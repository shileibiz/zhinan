## 2. 集成方案

### 2.1 就业率数据 ✅ (已完成)
- **数据源**: 麦可思《中国本科生就业报告》公开摘要
- **采集方法**: 内建数据（60个热门专业 × 3个年份）
- **入库策略**: 年更（每年6月麦可思发布新报告后手动更新）
- **工作量**: 小
- **状态**: ✅ 已完成

### 2.2 行业薪资统计 (industry_stats)
- **数据源**: 
  - 麦可思《就业蓝皮书》行业薪资表（摘要免费，全文付费）
  - 国家统计年鉴（免费）
  - 上市公司研报（东方财富/雪球免费看摘要）
- **采集方法**: 
  - 短期：内建数据（已知主要行业平均薪资）
  - 长期：HTTP采集 + PDF解析
- **入库策略**: 年更
- **工作量**: 小(内建) / 中(自动采集)
- **优先级**: 中

### 2.3 各省一分一段表
- **数据源**: 各省教育考试院官网（31个省份）
- **采集方法**: HTTP + HTML/PDF解析
- **入库策略**: 年更（每年高考放榜后）
- **工作量**: 大（31省格式各异）
- **优先级**: 高

### 2.4 高校就业质量报告PDF
- **数据源**: 每所高校官网 → 信息公开/招生就业栏目
- **覆盖范围**: 头部200校（985/211 + 热门双非）
- **采集方法**: HTTP下载PDF → PyMuPDF/pdfplumber解析
- **入库策略**: 年更（各校每年12月-次年3月发布）
- **工作量**: 大（各校格式不统一，需逐校编写解析器）
- **优先级**: 中

### 2.5 教育部统计公报 + 学生数分省表
- **数据源**: 
  - `http://www.moe.gov.cn/jyb_sjzl/sjzl_fztjgb/`（统计公报）
  - 教育部信息公开栏 → 高等教育学生数分省/学科表
- **采集方法**: HTTP + HTML/PDF解析
- **入库策略**: 年更（每年7月发布）
- **工作量**: 小
- **优先级**: 中

### 2.6 招生计划
- **数据源**: 阳光高考平台 `https://gaokao.chsi.com.cn`
- **采集方法**: HTTP + HTML解析
- **入库策略**: 全量覆盖
- **工作量**: 中（2000+高校 × 多省份）
- **优先级**: 高（推荐引擎核心依赖）

### 2.7 付费API加速（可选）
如果需要加速数据建设，以下付费API可作为补充：

| API | 数据内容 | 适用场景 | 费用参考 |
|:----|:---------|:---------|:---------|
| **咕咕数据** | 高校基础信息、历年录取分数、专业招生计划 | 系统集成，补录取分数/招生计划 | 按量付费 |
| **马克数据网** | 高校/专业录取分数历史数据 | 补历史分数趋势 | 按量付费 |

| **Wind/Choice** | 券商研报中教育/高校相关数据 | 趋势判断参考 | 机构订阅（个人贵） |

## 4. 已评估但排除的数据源

### 4.1 上市公司研报（教育科技/职教/招聘板块）
- **结论**：不作为数据采集通道
- **原因**：研报数据为行业级市场空间测算，无法拆解到"校-专业"颗粒度；
  底层一手数据（教育部公报/阳光高考/麦可思）已被第2节覆盖，研报只是二次转译
- **可能用途**：人工调研背景材料，辅助判断哪些专业/赛道值得优先采集
- **获取渠道**：东方财富/雪球/同花顺（免费）；Wind/Choice（付费）

### 4.2 百度指数 / 微信指数
- **结论**：不作为数据采集通道
- **原因**：无公开API，需模拟登录，维护成本高；新榜已覆盖微信生态热点

## 5. 优先级建议

### 高优先级（1-2个月）
1. ✅ 就业率采集器（已完成）
2. 招生计划采集（阳光高考）
3. 各省一分一段表采集（考试院）
4. 教育部统计公报 + 学生数分省表自动采集

### 中优先级（3-6个月）
5. 行业薪资统计（内建数据）
6. 高校排名接入（软科/校友会）
7. 高校就业质量报告 PDF 解析（头部50校）
8. 考虑接入咕咕数据/Mark数据付费API补齐历史分数

### 低优先级（6-12个月）
9. 全部200校 PDF 解析
10. 麦可思完整报告数据购买/接入
11. 自动增量更新机制
12. 行业数据扩展（部委/行业协会）

### 更新频率
- **教育数据**：每年更新（统计公报7月、高考录取7-8月、就业报告12月-次年3月）
- **行业数据**：按需更新
- **法律提醒**：商业使用注意数据版权；批量抓取遵守 robots 协议
