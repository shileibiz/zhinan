## 2. 集成方案

### 2.1 就业率数据 ✅ (本次完成)
- **数据源**: 麦可思《中国本科生就业报告》公开摘要
- **采集方法**: 内建数据（60个热门专业 × 3个年份）
- **入库策略**: 年更（每年6月麦可思发布新报告后手动更新）
- **工作量**: 小
- **状态**: ✅ 已完成

### 2.2 行业薪资统计 (industry_stats)
- **数据源**: 麦可思《就业蓝皮书》行业薪资表 / 国家统计年鉴
- **采集方法**: 
  - 短期：内建数据（已知主要行业平均薪资）
  - 长期：HTTP采集 + PDF解析
- **入库策略**: 年更
- **工作量**: 小(内建) / 中(自动采集)
- **优先级**: 中

### 2.3 各省一分一段表
- **数据源**: 各省教育考试院官网（31个省份）
- **URL示例**:
  - 北京: `https://www.bjeea.cn`
  - 广东: `https://eea.gd.gov.cn`
  - 上海: `https://www.shmeea.edu.cn`
- **采集方法**: HTTP + HTML/PDF解析
- **入库策略**: 年更（每年高考放榜后）
- **工作量**: 大（31省格式各异）
- **优先级**: 高

### 2.4 高校就业质量报告PDF
- **数据源**: 每所高校官网 → 信息公开/招生就业栏目
- **覆盖范围**: 头部200校（985/211 + 热门双非）
- **采集方法**: HTTP下载PDF → PyMuPDF/pdfplumber解析
- **入库策略**: 年更（各校每年12月-次年3月发布）
- **工作量**: 大（各校格式不统一，需逐校编写解析器）
- **优先级**: 中

### 2.5 教育部统计公报
- **数据源**: `http://www.moe.gov.cn/jyb_sjzl/sjzl_fztjgb/`
- **采集方法**: HTTP + HTML解析（统计公报为网页/PDF格式）
- **入库策略**: 年更（每年7月发布）
- **工作量**: 小
- **优先级**: 中

### 2.6 招生计划
- **数据源**: 阳光高考平台 `https://gaokao.chsi.com.cn`
- **采集方法**: HTTP + HTML解析
- **入库策略**: 全量覆盖
- **工作量**: 中（2000+高校 × 多省份）
- **优先级**: 高（推荐引擎核心依赖）

## 3. 优先级建议

### 高优先级（1-2个月）
1. ✅ 就业率采集器（本次完成）
2. 招生计划采集（阳光高考）
3. 各省一分一段表采集（考试院）
4. 教育部统计公报自动采集

### 中优先级（3-6个月）
5. 行业薪资统计（内建数据）
6. 高校排名接入（软科/校友会）
7. 高校就业质量报告 PDF 解析（头部50校）

### 低优先级（6-12个月）
8. 全部200校 PDF 解析
9. 麦可思完整报告数据购买/接入
10. 自动增量更新机制
